Descarga LucidFence (gratis, on-prem, $0)

Tu geofencing corre en tu máquina. Tus dispositivos, geocercas, riesgos y CVE se calculan ahí. Nada se envía a nuestra infra. Instala en un comando:

brew install adrimg3196/lucidfence/lucidfence Descargar v1.0.1 Ver repo

Si tu versión de Homebrew no resuelve el tap implícito de 3 partes, dos lineas: brew tap adrimg3196/lucidfence y luego brew install lucidfence.

Tras instalar: lucidfence serve y abre http://localhost:8765. La demo de arriba es datos de ejemplo; la tuya vive en tu infra.

¿Quieres ver la demo multi-tenant en vivo? La vitrina de arriba se regenera en la nube cada 15 min con datos de ejemplo. Para tu flota real, descarga e instala — no necesitas registrarte con nosotros.
